# 通用文案拆解+仿写对标万能框架
**全品类适配（短视频/小红书/公众号/朋友圈/带货文案）**
核心逻辑：**先拆透「原文案的骨架+血肉+灵魂」，再按「换皮→换肉→换魂」三级仿写，做到对标不抄袭、复用有逻辑**

---

## 使用注意事项
1. 仿写前必须先完整拆解对标文案结构，至少看清开头钩子、主体分段、结尾收口、情绪推进和转化动作。
2. 仿写时要严格对齐原文案的结构和大致字数，不能只改几个词就直接输出。
3. 如果对标文案是口播型内容，先拆结构，再结合转写文本还原节奏，最后按相同结构重写。
4. 输出仿写稿时，要保证内容原创、表达自然，但结构逻辑和篇幅控制要与对标样本一致。

---

## 一、万能文案拆解框架（三层标准化拆解）
### 核心原则：从外到内，从形到神，不漏任何关键元素
### 第一层：表层拆解·形式层（看得见的外壳）
**核心元素：结构、节奏、话术、格式**
1. **结构框架**
    - 开头：钩子类型（痛点提问/反常识/反差/悬念/数据）
    - 主体：段落分工（分点/递进/对比/故事）
    - 结尾：收尾类型（总结/互动/引导/转化/升华）
2. **节奏节奏**
    - 长短句比例、段落疏密、信息密度（快节奏/慢节奏）
3. **话术风格**
    - 语气（犀利/温柔/专业/口语/吐槽）
    - 句式（设问/反问/排比/短句爆破）
    - 用词（网感词/行业词/大白话/金句）
4. **格式排版**
    - 分段、符号、重点标注、段落长度

### 第二层：中层拆解·内容层（摸得着的血肉）
**核心元素：主题、逻辑、论据、案例**
1. **核心主题**
    - 文案核心观点/核心卖点/核心诉求（一句话概括）
2. **叙事逻辑**
    - 推进线：问题→原因→方案 / 过去→现在→未来 / 错误→正确 / 痛点→爽点
    - 论证方式：分点论证/对比论证/故事论证/原理论证
3. **论据支撑**
    - 数据、常识、原理、行业规律
4. **案例素材**
    - 自身案例/他人案例/场景案例/反面案例

### 第三层：深层拆解·内核层（抓得住的灵魂）
**核心元素：情绪、人群、人设、目的**
1. **情绪钩子**
    - 击中情绪：焦虑/共鸣/好奇/爽感/危机感/虚荣心
2. **目标人群**
    - 精准画像（身份+痛点+需求）
3. **账号人设**
    - 说话人设（导师/朋友/过来人/专家/吐槽者）
4. **最终目的**
    - 引流/转化/点赞收藏/认同观点/塑造认知

## 二、文案核心元素总表（对标速查版）
| 元素类别 | 核心拆解要点 | 仿写对标关键 |
|----------|--------------|--------------|
| **结构类** | 开头钩子、主体分段、结尾引导 | 照搬结构，不照搬文字 |
| **逻辑类** | 论证顺序、核心观点、推导路径 | 复用逻辑，替换内容 |
| **情绪类** | 痛点、爽点、共鸣点、焦虑点 | 对齐情绪，换场景触发 |
| **表达类** | 语气、句式、用词、金句 | 模仿风格，改写话术 |
| **信任类** | 案例、数据、亲身经历、权威背书 | 替换同类素材，增强可信度 |
| **转化类** | 行动指令、引流话术、变现引导 | 适配自身业务，修改引导语 |

## 三、万能仿写框架（三级仿写策略·全水平适配）
### 核心原则：**结构照搬、逻辑复用、情绪对齐、内容原创**
### 一级仿写·新手版（换皮不换骨·最快出稿）
**只替换表层元素，结构/逻辑/情绪完全照搬**
✅ 可替换元素：
1. 行业/领域：健身→护肤→AI→职场→育儿
2. 人群身份：打工人→宝妈→创业者→学生
3. 案例素材：他人故事→自身经历→行业案例
4. 关键词：专业术语、产品名、场景词
5. 金句改写：保留句式，替换核心词汇

### 二级仿写·进阶版（换骨不换魂·自然不生硬）
**保留结构+情绪，优化内容逻辑，改写表达**
✅ 可调整元素：
1. 调整论证顺序：分点→对比→故事
2. 替换切入角度：焦虑→共鸣→好奇
3. 优化话术风格：口语化→专业感→温柔风
4. 升级案例：普通案例→独家实操案例
5. 微调节奏：快节奏→慢节奏，增加细节

### 三级仿写·高手版（融魂成己·原创度拉满）
**只复用底层逻辑+情绪内核，内容完全重构**
✅ 核心保留：
- 核心观点逻辑、情绪触发点、目标人群
✅ 完全重写：
- 所有文字、案例、论据、表达、金句
- 加入自身独家经验/方法论/实操路径
- 形成专属个人风格，无对标痕迹

## 四、仿写万能替换公式（直接套用）
1. **开头钩子公式**
原文钩子逻辑 + 自身行业/人群/场景 = 仿写开头
例：反常识钩子→「上班越厉害，创业越难」→仿写「AI用越熟，AI变现越难」

2. **主体论证公式**
原文分点逻辑 + 自身领域痛点/方法 = 仿写主体
例：原文3点上班思维→仿写3点AI工具人思维

3. **结尾转化公式**
原文引导逻辑 + 自身业务/诉求 = 仿写结尾
例：原文引导咨询→仿写引导关注/领取AI资料

4. **金句仿写公式**
原文句式结构 + 自身核心观点 = 原创金句
例：「和不确定性共处，才是创业真相」→「和AI工具共生，才是一人公司真相」

看结构、理逻辑、抓情绪、拆话术、找案例、定目的
