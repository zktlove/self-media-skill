# 开头核心哲学

> 适用范围：短视频口播、观点表达、故事表达、知识表达。

## 核心信念

- 写不出好开头，很多时候不是开头技巧不够，而是内容本身没有价值、没有素材、没有冲击力。
- 开头是内容的试用装，不是标题的延续。
- 开头必须在前几秒独立建立吸引力，不能默认用户已经看过标题和封面。

## 开头基础公式

```text
开头 = 话题（讲什么）+ Hook（为什么继续看）+ 可信度（为什么信你）
```

## 四条基础原则

### 1. 先把话题说清楚

- 开头要让用户立刻知道你在讲什么。
- 不要只抛半句话，也不要把关键信息留给标题补全。

### 2. Hook 要能抓住继续看的理由

- 可以用结果、数据、反差、痛点、争议、悬念做抓手。
- Hook 不是堆词，而是给用户一个继续听下去的理由。

### 3. 可信度必须快速建立

- 可以是成绩、经历、观察结果、案例、身份、行业经验。
- 可信度越具体，越容易撑起后面的表达。

### 4. 必须口播友好

- 优先直接陈述，不要写成书面分析。
- 少用自问自答、长定语、抽象概念堆叠。
- 写完后要能一口气念出来。

## 创作提醒

- 不要把开头写成标题复述。
- 不要在开头直接把答案讲完，悬念要留到正文里兑现。
- 不要为了制造高级感而牺牲口语感。
