# Xiaohongshu Copywriting Guide

这个文件只服务第二步：把第一步收集到的对标内容，整理成可直接发布的小红书标题和正文。

## 第二步的目标

- 不是复述对标内容
- 不是拼接热门句子
- 是基于对标内容，产出一篇更适合当前账号定位的小红书文案

## 输入来源

第二步至少读取以下信息：

1. 第一步产出的 3 篇对标内容
2. 第一步生成的 Markdown 汇总文件
3. 用户提供的账号定位
4. 用户提供的目标人群
5. 用户提供的内容方向
6. 用户提供的语气风格

## 生成规则

1. 先提炼对标内容里的共同高频点
- 标题写法
- 切入角度
- 情绪点
- 结果承诺
- 结构节奏

2. 再结合用户自己的定位做改写
- 保留有效结构
- 不照抄原文
- 不直接搬运原句
- 不制造和定位无关的内容

3. 标题要符合小红书表达
- 尽量具体
- 尽量口语化
- 能让目标人群立刻知道“这篇和我有什么关系”

4. 正文要按“小红书图文可读性”来写
- 开头先点题
- 中间给出清晰信息点
- 结尾做轻收束或轻互动
- 不写成公众号长文

## 输出要求

第二步至少产出两项：

1. 最终标题
2. 最终正文

如果需要给第三步生图用，建议同步整理出一份 `copy_text`，格式如下：

```text
标题：{最终标题}
正文：{最终正文}
```

## 使用提醒

1. 第二步是承上启下的一步，既要吃掉第一步的对标内容，也要给第三步提供稳定的 `copy_text`。
2. 如果用户没有明确给出账号定位、目标人群、内容方向、语气风格，要先根据上下文补齐，再继续生成。
3. 如果第一步内容明显偏题，不要硬写，先指出问题，再决定是否重找对标内容。
